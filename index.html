Tutorial do Senova
Aprenda como usar cada módulo em poucos minutos.
🏠 1. Home
Seu cockpit diário: KPIs de candidaturas, próximas ações com prazo, emails do Outlook e alertas de mercado — tudo num só lugar.
🎯 2. Análise CV
Cole a descrição de uma vaga e receba seu CV otimizado com as palavras-chave certas + score ATS em segundos.
📋 3. Pipeline
Kanban de candidaturas com 5 colunas. Arraste cards entre estágios, defina próximas ações e acompanhe contatos do CRM.
👤 4. Perfil
Configure cargo alvo, idiomas, plataformas onde está cadastrado e score mínimo por região — esses dados alimentam a varredura automática.
🔍 5. Busca de Vagas
Varredura automática diária às 07h BRT via Adzuna + Jobicy (BR · ES · DE · PT · Remoto). Importe as oportunidades para o Pipeline com um clique.
🔄 6. Fluxo Completo
O caminho ideal: Perfil → Varredura → Análise CV → Pipeline → Candidatura via Outlook → Follow-up automático em 7 dias. Cada etapa registra uma entrada no histórico da vaga.
