# SKILL: Pesquisa de Executivo — Mapeamento para Abordagem

## Quando usar
Sempre que Marcos precisar:
- Identificar quem lidera uma área em empresa-alvo
- Verificar cargo atual de um executivo
- Mapear conexões em comum no LinkedIn
- Preparar contexto antes de enviar convite ou mensagem
- Detectar movimentação executiva (promoção, saída, chegada)

## Processo padrão
1. Buscar nome + empresa + cargo na web (LinkedIn público, Google, notícias)
2. Verificar: cargo atual confirmado? Data da última atualização?
3. Identificar: trajetória relevante, empresas anteriores notáveis
4. Verificar conexões em comum com Marcos (via LinkedIn público se disponível)
5. Identificar ganchos de abordagem: prêmios, artigos, expansões, anúncios recentes
6. Classificar temperatura do contato: Frio / Morno / Quente

## Output padrão
- Nome completo + cargo atual confirmado
- Empresa + setor
- Trajetória resumida (3–5 pontos relevantes)
- Ganchos de abordagem identificados
- Temperatura sugerida: Frio / Morno / Quente
- Sugestão de próximo passo (convite LinkedIn / mensagem direta / via indicação)

## Contexto de Marcos para cruzar
- Território natural: mídia, TV, educação, tecnologia, marketing, expansão
- Empresas de alto interesse: grupos de comunicação regionais/nacionais, fintechs em expansão, scale-ups com captação recente, empresas europeias chegando ao Brasil
- Conexões estratégicas já mapeadas: RPC/GRPCOM, NSC, RBS, RIC, Globo Nacional

## Alerta de privacidade
- Nunca incluir dados pessoais sensíveis (CPF, endereço, telefone pessoal)
- Usar apenas informações públicas
