# SKILL: Análise Concorrencial — Produtos e Funcionalidades

## Versão 1.0 — mai/2026

## Identidade
Analista sênior de produto e inteligência competitiva. Avalia concorrentes com visão de:
- **Produto** — funcionalidades, UX, fluxos, diferenciais
- **Negócio** — modelo de receita, precificação, posicionamento
- **Mercado** — segmento atendido, gaps, oportunidades
- **Tecnologia** — stack, integrações, extensões, APIs

## Quando usar
SEMPRE antes de:
- Propor uma nova funcionalidade para o Senova
- Definir precificação
- Atualizar o roadmap
- Preparar pitch ou business plan
- Comparar o Senova com o mercado

Nunca propor algo novo sem antes verificar se já existe no mercado e como os concorrentes resolvem.

## Concorrentes Mapeados (mai/2026)

### 1. Teal (tealhq.com)
**Foco:** ATS + tracking + IA integrada
**Preço:** Grátis (15 créditos) / $29/mês Pro
**Público:** Profissionais em geral, foco em volume
**Pontos fortes:**
- Chrome extension para salvar vagas de 40+ job boards
- ATS score em tempo real vs. descrição da vaga
- Resume builder com keyword matching
- Networking CRM básico
- Tracking ilimitado no plano grátis
**Pontos fracos:**
- Interface menos visual (lista, não kanban)
- Sem foco em executivos 50+
- IA como add-on caro, não nativa
- Sem simulador de entrevista integrado
**Aprendizado para o Senova:** extensão Chrome é o diferencial de captura

### 2. Huntr (huntr.co)
**Foco:** Kanban visual + CRM de contatos
**Preço:** Grátis (40 vagas) / $30/mês Pro
**Público:** Buscadores de emprego que pensam como vendedores
**Pontos fortes:**
- Kanban visual superior — melhor do mercado
- CRM de contatos integrado ao pipeline
- Activity reminders para follow-ups
- Interface limpa, baixa curva de aprendizado
- Chrome extension com autofill
**Pontos fracos:**
- Sem IA de resume/ATS
- Sem simulador de entrevista
- Precisa combinar com Jobscan para ATS
- Sem foco executivo 50+
**Aprendizado para o Senova:** modelo de CRM de contatos é referência

### 3. Careerflow (careerflow.ai)
**Foco:** LinkedIn optimization + ATS + tracking
**Preço:** Grátis (10 vagas) / $14–24/mês
**Público:** Profissionais LinkedIn-heavy
**Pontos fortes:**
- LinkedIn profile reviewer com IA
- Resume optimizer com edições lado a lado
- Salva vagas em 1 clique
- Cover letter com IA
**Pontos fracos:**
- Fortemente dependente do LinkedIn
- Outputs de IA precisam revisão constante
- Sem simulador de entrevista
- Sem foco 50+
**Aprendizado para o Senova:** cover letter com IA é feature esperada

### 4. Jobscan (jobscan.co)
**Foco:** ATS keyword matching
**Preço:** 5 scans grátis / $30+/mês
**Público:** Quem quer maximizar aprovação em ATS
**Pontos fortes:**
- Análise detalhada de keywords por seção
- Scan de resume vs. JD com score
- Cover letter optimizer
**Pontos fracos:**
- Não é tracker/CRM
- Accuracy inconsistente nos testes
- Sem IA nativa completa
- Caro para o que entrega
**Aprendizado para o Senova:** score ATS com % é feature valorizada

### 5. Simplify (simplify.jobs)
**Foco:** Autofill em portais de vagas
**Preço:** Grátis
**Público:** Candidatos de alto volume (10+ aplicações/dia)
**Pontos fortes:**
- Autofill em 100+ portais (Workday, Greenhouse, iCIMS, Lever)
- Grátis
- Salva tempo em formulários repetitivos
**Pontos fracos:**
- Tracker é secundário
- Sem IA de resume
- Sem CRM
**Aprendizado para o Senova:** autofill não é prioridade para executivos 50+

### 6. JobShinobi (jobshinobi.com)
**Foco:** Email → CRM automático via IA
**Preço:** ~$10/mês
**Público:** Quem odeia atualização manual
**Pontos fortes:**
- Forward email → cria/atualiza entrada no CRM automaticamente
- Fuzzy matching para evitar duplicatas
- Foco em eliminar overhead manual
**Pontos fracos:**
- Sem extensão Chrome
- Menor comunidade
- Sem resume builder
**Aprendizado para o Senova:** o Worker v6 já tem 80% desta funcionalidade — só falta ativar

### 7. Prentus (prentus.com)
**Foco:** Tracking + mock interviews com voz
**Preço:** Grátis
**Público:** Candidatos que precisam praticar entrevistas
**Pontos fortes:**
- Mock interviews com perguntas reais da vaga
- Resume builder com IA
- Cover letter generator
- Tudo grátis
**Pontos fracos:**
- Menos maduro que Teal/Huntr
- Mock interview só voz (sem análise de resposta)
**Aprendizado para o Senova:** simulador com perguntas da vaga específica é diferencial

### 8. JibberJobber (jibberjobber.com)
**Foco:** CRM customizável para networking
**Preço:** Grátis / plano vitalício disponível
**Público:** Profissionais que priorizam networking sobre volume
**Pontos fortes:**
- Altamente customizável
- Foco em relacionamentos de longo prazo
- Métricas customizadas
**Pontos fracos:**
- Interface datada
- Sem IA
- Curva de aprendizado alta
**Aprendizado para o Senova:** networking de longo prazo é relevante para executivos 50+

## Gaps do Mercado — Oportunidades para o Senova

1. **Foco 50+** — nenhum concorrente atende este segmento
2. **PT/BR nativo** — mercado brasileiro completamente descoberto
3. **IA nativa acessível** — concorrentes cobram $29–30 por IA; Senova a R$47
4. **Email → CRM** — só JobShinobi tem; Senova já tem estrutura no Worker
5. **Cover letter IA** — todos têm; Senova ainda não (prioridade)
6. **Chrome extension** — todos têm; Senova ainda não (médio prazo)
7. **Mock interview com análise** — Prentus tem voz, mas sem análise de resposta; Senova pode superar

## Processo de análise concorrencial recorrente
1. Pesquisar novidades dos concorrentes a cada 30 dias
2. Verificar novas features lançadas
3. Atualizar tabela comparativa
4. Identificar gaps que o Senova deve fechar
5. Priorizar no roadmap conforme valor e esforço
